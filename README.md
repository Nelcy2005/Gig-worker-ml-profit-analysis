# Gig Worker Profit Analysis using Machine Learning

## Overview
This project analyzes delivery worker earnings and predicts profit or loss based on working conditions such as petrol cost, strike days, and deliveries.

## Techniques Implemented
- Data Exploration (Parametric Methods)
- Regression Analysis
- Principal Component Analysis (PCA)
- Multi-Dimensional Scaling (MDS)
- Linear Discriminant Analysis
- Logistic Regression
- Kernel Machines (SVM)
- Multi-Layer Perceptron (MLP)
- Ensemble Learning (Random Forest)
- K-Means Clustering
- Hierarchical Clustering

## Interactive UI
A Gradio interface allows users to enter:
- Earnings
- Petrol cost
- Penalty

The system calculates:
- Net income
- Profit rate
- Profit or loss

## Technologies
Python, Pandas, Scikit-learn, Gradio

## Run the project
python app.py
